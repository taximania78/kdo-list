# Politique de sécurité

## Signaler une vulnérabilité

Si vous découvrez une faille de sécurité dans Kdo List, **merci de ne pas ouvrir d'issue publique**.

Utilisez le signalement privé de GitHub :
[Security → Report a vulnerability](https://github.com/taximania78/kdo-list/security/advisories/new)

Décrivez le problème, les étapes de reproduction et l'impact estimé. Une réponse sera apportée sous 7 jours.

## Versions supportées

Seule la dernière version (branche `main`) reçoit des correctifs de sécurité.

| Version | Supportée |
| ------- | --------- |
| `main`  | ✅        |
| autres  | ❌        |
